Install TaskVantage App

Get the best experience with our Progressive Web App

Installation Instructions

Choose your platform below:

computer Windows
language Google Chrome
  1. Open TaskVantage in Google Chrome
  2. Look for the install icon (install_desktop) in the address bar (right side)
  3. Click the install icon or the "Install" button
  4. Click "Install" in the popup dialog
  5. The app will open in its own window and appear in your Start Menu
info

Alternative method: Click the three dots menu (⋮) → "Save and share" → "Install TaskVantage..."

 explore Microsoft Edge
  1. Open TaskVantage in Microsoft Edge
  2. Look for the app available icon (add_circle) in the address bar
  3. Click the icon and select "Install"
  4. Or click the three dots menu (...) → "Apps" → "Install this site as an app"
  5. Click "Install" in the confirmation dialog
  6. TaskVantage will open as a standalone app and be pinned to your taskbar
 whatshot Mozilla Firefox
  1. Open TaskVantage in Firefox
  2. Click the three horizontal lines menu (☰) in the top right
  3. Select "Install TaskVantage" or "Add to Home Screen"
  4. Click "Install" in the popup
  5. The app will be added to your applications
warning

Note: Firefox PWA support may be limited. Chrome or Edge recommended for best experience.

help_outline Don't see the install option?

  • Make sure you're using the latest version of your browser
  • The install prompt may only appear after visiting the site a few times
  • Check if TaskVantage is already installed (look in Start Menu or Apps list)
  • Try refreshing the page
 phone_iphone iPhone
info

On iPhone, you must use Safari browser to install the app. Chrome and other browsers don't support PWA installation on iOS.

Safari Installation Steps:

  1. Open TaskVantage in Safari

    Navigate to your TaskVantage URL in the Safari browser

  2. Tap the Share button

    Look for the ios_share icon (square with arrow pointing up) at the bottom center or top right of Safari

  3. Scroll down and find "Add to Home Screen"

    You may need to scroll down in the share menu to see this option

  4. Tap "Add to Home Screen"

    You can customize the app name if desired

  5. Tap "Add" in the top right

    The TaskVantage icon will appear on your home screen

  6. Launch the app from your home screen

    Tap the TaskVantage icon to open it like a native app

help_outline Troubleshooting

  • Share button not visible? Make sure you're in Safari, not Chrome or another browser
  • Don't see "Add to Home Screen"? Update iOS to the latest version (iOS 11.3 or later required)
  • App won't open? Try removing it from home screen and re-adding it
tips_and_updates

Pro tip: You can organize the TaskVantage icon into folders just like any other app on your iPhone!

 phone_android Android
info

Google Chrome provides the best PWA experience on Android.

Chrome Installation Steps:

  1. Open TaskVantage in Chrome

    Navigate to TaskVantage in the Chrome browser on your Android device

  2. Look for the install prompt

    A banner may appear at the bottom saying "Add TaskVantage to Home screen" - tap "Add" or "Install"

  3. If no prompt appears, use the menu

    Tap the three dots menu (⋮) in the top right corner

  4. Select "Add to Home screen" or "Install app"

    The wording may vary depending on your Android version

  5. Confirm the installation

    Tap "Add" or "Install" in the confirmation dialog

  6. Launch from your home screen

    The TaskVantage icon will appear on your home screen and app drawer

phone_android Samsung Internet Browser
  1. Open TaskVantage in Samsung Internet
  2. Tap the menu icon (three lines) at the bottom
  3. Select "Add page to" → "Home screen"
  4. Tap "Add" to confirm
 language Other Browsers (Firefox, Opera, etc.)
  1. Open TaskVantage in your browser
  2. Look for the browser's menu (usually three dots or lines)
  3. Look for options like "Add to Home screen" or "Install"
  4. Follow the prompts to add the app

help_outline Troubleshooting

  • Install option missing? Make sure you're using Chrome 40+ or a compatible browser
  • Icon not appearing? Check your home screen and app drawer - it may take a moment
  • App won't open? Clear Chrome cache and try reinstalling
  • Banner disappeared? Use the three dots menu method instead
tips_and_updates

Pro tip: Once installed, TaskVantage will update automatically in the background. You'll always have the latest version!
